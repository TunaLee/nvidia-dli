# 실습_시작하기.ipynb 용 스크린샷

아래 5개 파일명으로 이 폴더에 저장한 뒤, 리포 루트에서:
```bash
python embed_screenshots.py
```
을 실행하면 `실습_시작하기.ipynb` 안에 이미지가 **base64로 완전히 박혀** 자체 완결 노트북이 됩니다.
(학생은 그 노트북 한 파일만 열면 됨)

| 파일명 | 화면 |
|---|---|
| `nvidia-account-menu.png` | 계정 아바타 메뉴 (API Keys 진입 + `Up to 40 rpm`) |
| `nvidia-apikeys-empty.png` | API Keys 빈 화면 + `Generate API Key` |
| `nvidia-generate-dialog.png` | 키 생성 다이얼로그 (Key Name / Never Expire) |
| `nvidia-phone-verify.png` | 전화번호(OTP) 인증 화면 |
| `nvidia-key-granted.png` | 키 발급 완료 (Copy API Key) |
