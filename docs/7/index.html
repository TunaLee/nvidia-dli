CH4연결

우리 루프 ↔ LangGraph 대응

→
model 노드 = 우리 ‘판단’(llm.chat)
→
tools 노드 = 우리 ‘행동/관찰’(call_tool)
→
tools→model 엣지 = 우리 루프(사이클)
→
조건부 엣지 = 도구 부르면 계속, 아니면 끝
graph.py
g = StateGraph(State)
g.add_node("model", call_model)   # 판단
g.add_node("tools", call_tools)   # 행동
g.add_conditional_edges("model", cont,
    {"tools": "tools", "end": END})
g.add_edge("tools", "model")      # 사이클!
타이밍
바닥부터 짜본 다음에 프레임워크를 보는 것이 순서다.
