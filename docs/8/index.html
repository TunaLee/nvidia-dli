→

CH4구성

무엇이 컨텍스트를 채우나

요소원칙

도구기능 중복 없이, 토큰 효율적 결과
예시(few-shot)엣지 나열 말고 대표 사례로
히스토리동적으로 필요한 부분만
검색 결과필요할 때만(just-in-time) 로드
한 줄
각 요소가 윈도우를 두고 경쟁한다 — 신호 대비 토큰을 따진다.
