## 공통 — 덱 편집 규약

모든 덱 편집은 아래 헬퍼 패턴을 쓴다. 슬라이드는 `data-title` 로 찾는다.

```python
import re
DOC = "docs/8/index.html"

def load():
    doc = open(DOC, encoding="utf-8").read()
    parts = re.split(r'(?=<section class="slide)', doc)
    return parts[0], parts[1:]          # head, sections

def title_of(sec):
    m = re.search(r'data-title="([^"]*)"', sec)
    return m.group(1) if m else ""

def find(secs, title):
    for i, s in enumerate(secs):
        if title_of(s) == title:
            return i
    raise AssertionError(f"슬라이드 못 찾음: {title}")

def save(head, secs):
    deck = head + "".join(secs)
    assert deck.count("<section") == deck.count("</section>"), "section 불균형"
    assert deck.count("<div") == deck.count("</div>"), "div 불균형"
    open(DOC, "w", encoding="utf-8").write(deck)
    print("슬라이드:", deck.count('<section class="slide'))
```

**검증 명령** (모든 덱 태스크 후 실행):
