CH2학습 설정

학습률 · 에폭 · 배치

보폭

학습률(lr)

한 걸음 크기. 너무 크면 골짜기를 건너뛰고, 너무 작으면 한없이 느리다.

감각

CH3비선형의 힘

활성화 함수 — 왜 ‘비선형’이 필요한가

기본

ReLU

음수는 0, 양수는 그대로. 빠르고 잘 작동 — 은닉층 표준.

확률

Sigmoid

0~1로 압축. 이진 분류 출력에.

다중분류

Softmax

여러 클래스를 합=1 확률로. 출력층에.

핵심

비선형이 없으면 층을 아무리 쌓아도 ‘직선 하나’와 같다. 활성화 함수가 곡선·복잡한 경계를 가능하게 한다.

CH3일반화의 적

과적합 — ‘외워버린’ 모델

✕ 과적합(overfitting)

학습 데이터는 완벽히 맞히지만 새 데이터엔 실패. 답을 ‘외운’ 상태 — 시험 문제만 외운 학생.

✓ 적정 일반화

학습·검증 데이터 모두 고르게 잘 맞힘. ‘원리를 이해한’ 상태.

신호

학습 손실↓인데 검증 손실↑로 벌어지면 과적합. 반대로 둘 다 높으면 과소적합(덜 배움).

LAB 3학습

학습 루프 — 예측·손실·역전파

lab3_train.py
loss_fn = nn.CrossEntropyLoss()
opt = torch.optim.Adam(model.parameters(), lr=1e-3)

for epoch in range(5):
    for x, y in train_dl:
        x, y = x.to(device), y.to(device)
        pred = model(x)              # ① 순전파
        loss = loss_fn(pred, y)      # ② 손실
        opt.zero_grad(); loss.backward()  # ③ 역전파
        opt.step()                   # ④ 가중치 수정
    print(epoch, loss.item())
✅ CH2의 ‘예측→손실→역전파→수정’ 네 줄이 그대로 코드로
